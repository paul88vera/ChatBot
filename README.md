# ChatBox

An intelligent Ai chatbot that leverages GROK Ai that will answer client questions with professional company based answers.

## Technology Used

Front-End:

- React.js
- Axios
- React Icons
- Clerk-React

Back-End:

- GROK 3 Mini
- Express.js
- Node.js
- Nodemon
- Cors
- Dotenvx

Widget:

- React.js
- Amazon S3

## Features

- Enter Company information to use with Ai frequent questions
- Answers client questions using company knowledge
